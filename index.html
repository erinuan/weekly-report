系列 Campaign
一个系列下挂多个广告组
* 账户 ID
* 系列名称
* 预算模式
* 预算金额
无限预算填 0
🎯
广告组 Adgroup
每组配置相同，出价 + 定向 + 预算
* 组名前缀
后缀自动加 -1 -2 -3 …
* 结构比例
格式：系-组-广
例 1-3-4 = 3组×4素材
* 组日预算 ($)
* 转化出价 ($)
* 投放地域
如列表无需要的国家，选「其他」后手动输入 geo_id
* Pixel ID
* 发布者名称
填写发布者的显示名称（display_name），类型固定为 TT_USER
🖼️
广告 Ad
每个广告 = 1 张图片 + 文案 + 落地页
* 广告文案
* 落地页链接
可包含宏：__CLICKID__ __CSITE__ __AID__ __CID__ 等
* 素材 ID 列表
每行一个素材 ID · 当前已输入 0 个，需要 12 个
